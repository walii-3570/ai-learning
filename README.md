## ⚙️ Setup

Install dependencies based on what you want to run:

```bash
# Core
pip install numpy pandas matplotlib scikit-learn

# Deep Learning
pip install torch torchvision

# NLP & Embeddings
pip install sentence-transformers groq
```

> Note: Groq API key required for `chatbot.py` and `embedding.py`.  
> Add your key to `config.py` — never hardcode it in your files.
